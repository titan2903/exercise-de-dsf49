# Data Engineer Dibimbing

## Tugas Utama

1. Ambil dataset raw dari Kaggle, sumber lain, atau buat data dummy. Pilih data yang sesuai kemampuan dan cakupan pengerjaan.
2. Lakukan cleansing data atau data quality check. Kamu juga bisa melakukan filtering agar hanya data yang relevan yang digunakan.
3. Ekspor output ke Excel atau ke database. Database bebas: MySQL, PostgreSQL, BigQuery, atau lainnya.
4. Implementasikan standar data governance, misalnya masking untuk data sensitif dan membuat data dictionary dari dataset yang digunakan.
5. Presentasikan hasil serapi mungkin, bisa dalam bentuk PowerPoint atau Word.
6. Upload hasil ke LinkedIn, tag Dibimbing, dan gunakan hashtag dibimbing.

## Opsional

1. Jika data memungkinkan, lakukan analisis dasar dan buat chart/visualisasi dengan Python.
2. Tag LinkedIn: Keyvi Caisa Risman.
